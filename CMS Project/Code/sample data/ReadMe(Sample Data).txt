Author_notification.txt is for chair to notify author.

Chair_notification.txt is for notifying chair when all reviewers evaluate the paper.

conference.txt is used to store the conferences information.

paper.txt is used to store the paper information.

paperscore.txt is used to store the reviewers' evaluation.

Reviewer_notification.txt is for chair to notify reviewers to reviewer the papers.
 
topics.txt is used to store the topics.

userInfo.txt is used to store the users' information.
